‚¨á Basic Installation

  1. Instal Steam Tool yang sudah kalian download dan buka steam tool
  2. Ekstrak file game yang telah kalian unduh
  3. block/Pilih file hasil ekstraksi game tersebut
  4. Drag file yang dipilih ke logo steam tool yang mengambang di layar
  5. Restart Steam
